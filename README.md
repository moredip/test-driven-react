# How do I... ?

## Build & unit-test
```
npm install -g gulp
npm install
gulp
```

## Run all tests (unit- and browser-tests)
```
npm test
```

## Run the app
```
open build/index.html
```

## watch and continuously run build & unit-test
```
gulp watch
```

## watch and continuously run browser-tests
```
node_modules/.bin/testem -f tests/testem.json
```
